- ReactJS
- Redux
- TailwindCSS
- SCSS
- Recharts
- ViteJS
- jsPDF
#### Server-Side
- ExpressJS
- Mongoose
- JWT
- Google oauth
- BCrypt
- Cookie-parser
#### Database
- MongoDB
#### Deployment
- Netlify - Client
- Heroku - Server
- MongoDB Atlas - Database
